# Todo List App

A modern, responsive task management application built with Angular and Tailwind CSS. You can access it through the following link:

[https://kikilist.vercel.app](https://kikilist.vercel.app)

## Features

- **Task Management**: Create, read, update, and delete (CRUD) tasks easily.
- **Task Status**: Mark tasks as completed or pending with a simple click.
- **Views**: Filter tasks by "Current Tasks" or "Completed Tasks".
- **Bulk Actions**: "Clear All" button to remove all visible tasks at once.
- **Dark Mode**: Fully supported dark/light theme toggle with persistence.
- **Responsive Design**: Optimized for both desktop and mobile devices.
- **Data Persistence**: Tasks and theme preferences are saved in LocalStorage.
- **Modern UI**: Clean interface with smooth transitions, modal confirmations, and a "Scroll to Top" button.

## Tech Stack

- **Framework**: [Angular v21](https://angular.io/) (Standalone Components, Signals)
- **Styling**: [Tailwind CSS v4](https://tailwindcss.com/)
- **Language**: [TypeScript](https://www.typescriptlang.org/)

## Installation & Setup

1.  **Clone the repository**:
    ```bash
    git clone https://github.com/kikicodes23/todoList.git
    cd todoList
    ```

2.  **Install dependencies**:
    ```bash
    npm install
    ```

3.  **Run the application**:
    ```bash
    ng serve
    ```
    Navigate to `http://localhost:4200/`. The application will automatically reload if you change any of the source files.
